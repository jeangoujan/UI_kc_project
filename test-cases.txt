Test-case #1 | Заполнение формы корректными данными

Description:
Проверка отправки формы с валидными данными

Preconditions:
1) Пользователь находится на странице заполнения формы
(https://demoqa.com/automation-practice-form)

Scenario:
1) Ввести имя в поле 'First Name', например 'Филипп'
Expected result: Имя введено
2) Ввести фамилию в поле 'Last Name', например 'Киркоров'
Expected result: Фамилия введена
3) Ввести электронный адрес в поле 'Email', например 'philya1970@mail.ru'
Expected result: Email введен
4) Выбрать пол (Gender), нажатием на одну из радиокнопок, например 'Male'
Expected result: Пол выбран
5) Ввести номер телефона в поле 'Mobile', например '77476674747'
Expected result: Номер телефона введен
6) Выбрать дату рождения в поле 'Date of birth', например 5 июня 1970
Expected result: Дата рождения выбрана, отображается как 05 Jun,1970
7) Выбрать subjects, например English
Expected result: При вводе 'English' появляется возможность выбрать значение из списка, при нажатии, subject закрепляется
в поле Subjects
8) Выбрать хобби в Hobbies, поставив чекбокс у соответствующего хобби, например 'Sports, Reading'
Expected result: Выбраны 'Sports и Reading' - проставлены чекбоксы
9) Загрузить фотографию с компьютера нажав на кнопку 'Upload file'
Expected result: Открывается директория компьютера, для выбора фотографии, после выбора название фотографии
отображается у кнопки 'Upload file'
10) В поле 'Current Address' ввести адрес
Expected result: Адрес введен
11) Выбрать штат нажав на кнопку 'Select State'
Expected result: При клике на кнопку, появляется раскрывающийся список с выбором штатов (NCR, Uttar Pradesh, Haryana, Rajasthan)
При клике на штат, его название отображается в поле где было 'Select State'
12) Выбрать город нажав на кнопку 'Select City'
Expected result: При клике на кнопку, появляется раскрывающийся список с выбором городом (Delhi, Gurgaon, Noida)
При клике на город, его название отображается в поле где было 'Select City'
13) Нажать на кнопку 'Submit'
Expected result: Открывается модальное окно с данными, которые соответствуют всем введенным ранее данным

Description:
Проверка отправки формы с некорректными данными в необязательные поля

Preconditions:
1) Пользователь находится на странице заполнения формы
(https://demoqa.com/automation-practice-form)

Scenario:

1) Ввести имя в поле 'First Name', например 'Philipp'
Expected result: Имя введено
2) Ввести фамилию в поле 'Last Name', например 'Morris'
Expected result: Фамилия введена
3) Ввести некорректный электронный почтовый адрес в поле 'Email', например 'morris@@@mail.ru'
Expected result: Поле обрамляется красным цветом, подсказывая о некорректности введенного Email

Expected result: Невозможно выбрать даты будущего

Expected result: Некорректный subject введен, он не закрепляется в Subjects
8) Загрузить файл формата отличного от стандартных форматов изображений нажатием на кнопку 'Choose file'
Expected result: Невозможно загрузить файл, отличный от стандартных форматов изображений (jpg, png, raw)
9) Ввести адрес в поле 'Current Address'
Expected result: Адрес введен
10) Выбрать штат, нажатием на кнопку 'Select State'
Expected result: Открывается список со штатами
11) Выбрать город, нажатием на кнопку 'Select City'
Expected result: открывается список с городами
12) Нажать на кнопку 'Submit'
Expected result: Форма не отправляется, так как необязательные поля заполнены некорректно
